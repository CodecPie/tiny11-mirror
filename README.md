## Setup
- Download one of the ISO Files (b2 no sysreq - use this for old PCs/PCs that normally do not support Win11)
- Donwload https://rufus.ie/de/ - Enter the ISO, burn it to an USB Stick (16GB)
- Install Tiny-Windows11 :)
